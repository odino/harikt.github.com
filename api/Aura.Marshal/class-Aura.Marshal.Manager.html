protected Aura\Marshal\Type\GenericType
# buildType( string $name )

Builds a type object from a type definition.

The build process happens in two stages:

  1. Use the $type_builder to create the type object.
  2. Add relationships from the type definition.

The two-stage process helps avoid race conditions where a type may have relationships to other type object that might not be in the manager yet.
