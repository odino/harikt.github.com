# lastInsertId( string $table, string $col )

Returns the last ID inserted on the connection for a given table and column sequence.

PostgreSQL uses a sequence named for the table and column to track auto-incremented IDs; you need to pass the table and column name to tell PostgreSQL which sequence to check.
